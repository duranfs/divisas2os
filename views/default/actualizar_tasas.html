Actualización de Tasas

Haga clic en el botón para obtener las tasas más recientes del BCV.

Información

¿Qué hace esta función?

  • Consulta la página oficial del BCV
  • Extrae las tasas USD/VES y EUR/VES
  • Las guarda en la base de datos
  • Las marca como activas para uso inmediato

Después de actualizar:

  • Las calculadoras usarán las tasas reales
  • Los widgets mostrarán valores actualizados
  • Las transacciones usarán tasas oficiales
